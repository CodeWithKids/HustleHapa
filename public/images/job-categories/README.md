# Job Category Images

Place your job category images in this directory with the following naming convention:

- `mjengo.jpg` - Construction/Mjengo category image
- `babysitting.jpg` - Babysitting category image
- `laundry.jpg` - Laundry category image
- `carpentry.jpg` - Carpentry category image
- `gardening.jpg` - Gardening category image
- `cleaning.jpg` - Cleaning category image
- `default.jpg` - Fallback image (optional)

## Image Specifications

Recommended image specifications:
- **Size**: 200x200px to 400x400px (square)
- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: 1:1 (square) for best display
- **File Size**: Optimized for web (under 100KB per image recommended)

## Image Sources

You can use:
- Stock photos from Unsplash, Pexels, or Pixabay
- Custom illustrations
- Icons with backgrounds

## Fallback

If an image fails to load, the component will automatically fall back to the emoji icon for that category.

## Examples

Good image examples:
- Clear, recognizable representation of the job category
- Professional, friendly appearance
- Consistent style across all images
- High contrast for visibility on light backgrounds
